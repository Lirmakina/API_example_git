# Users API

Users API предоставляет доступ к управлению пользователями сервиса.  
API позволяет внешним системам получать данные о пользователях, создавать новых пользователей и обновлять существующие.

API предназначен для интеграций и автоматизации работы с пользователями.

---

## Что можно делать через API

С помощью Users API можно:

- получить список пользователей;
- получить информацию о конкретном пользователе;
- создать пользователя;
- изменить данные пользователя;
- удалить пользователя.

Все операции выполняются по HTTP.  
Данные передаются и возвращаются в формате JSON.

## Основные сущности

### Пользователь

Пользователь — это основная сущность API.  
Пользователь описывает человека, зарегистрированного в системе, и содержит его идентификатор, персональные данные и статус аккаунта.

---

## Основные сценарии использования

### Получение списка пользователей
Используется для синхронизации пользователей между системами и получения актуального списка пользователей сервиса.

### Работа с пользователем
Позволяет получать данные пользователя по идентификатору, изменять его данные или удалять пользователя.

### Создание пользователей
Используется для автоматического добавления пользователей во внешних системах.
