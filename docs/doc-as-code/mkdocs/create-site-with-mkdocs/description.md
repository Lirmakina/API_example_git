### Что вы сделаете с помощью инструкции

Мы создадим сайт с документацией, который будет доступен в интернете. 

Процесс создания сайта охватывает сразу несколько знаний:
- Создадим удаленный репозиторий Git и привяжем к нему локальный.
- Установим MkDocs, Python,...
- Создадим документацию в markdown.
- Настроим автопубликацию и выполним деплой. Это между прочим CI/CD.
